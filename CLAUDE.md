# FALCON プロジェクト Claude 指示書

## 妊娠率（21日PR）の大前提

**妊娠率に関する実装・議論・修正は、必ず以下を唯一の仕様書として扱うこと。**

```
reference/pregnancy_rate_research.md
```

このファイルは `Bredsum.xlsx`（農場実績データ）との全サイクル数値照合済みであり、
DairyComp305互換ロジックの正式仕様として確定している。

### 核心ルール（概要）

| 項目 | 仕様 |
|------|------|
| サイクル長 | 21日 |
| サイクル数 | 起算日から18サイクル |
| 有効サイクル条件 | `cycle_end + 28日 ≤ today`（未満は妊娠対象=0） |
| 繁殖対象牛 | DIM ≥ VWP かつ オープン かつ 非DNB かつ 在籍中（11日以上） |
| VWPデフォルト | 50日（農場設定可・本農場実績値は42日） |
| 妊娠対象牛 | 繁殖対象牛 − AI実施済みかつS=未確定の牛 |
| 妊娠数 | S=P（妊娠確認）+ S=A（受胎後流産） |
| 妊娠率計算式 | `round(妊娠数 / 妊娠対象牛数 × 100)` |

詳細なロジック・S値定義・退牧牛処理・サンプルコードは
`reference/pregnancy_rate_research.md` セクション3〜9を参照。

### FALCONイベント番号対応

| S値相当 | FALCONイベント番号 |
|---------|----------------|
| AI実施 | 200 (AI), 201 (ET) |
| S=P | 303 (PDP), 307 (PAGP) |
| S=O | 302 (PDN), 306 (PAGN) |
| S=A | 305 (ABRT) |
| DNB | 204 (STOPR) |
| SOLD | 205 |
| DEAD | 206 |
