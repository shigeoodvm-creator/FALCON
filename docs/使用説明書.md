## 1. アプリ概要

**FALCON2** は酪農家向けの牛群管理システムです。

- 個体ごとの繁殖・健康・乳量データをイベントとして記録
- 受胎率・乳量・リニアスコア等をリスト/集計/グラフで分析
- 繁殖検診表の作成・印刷
- 乳検データ・DC305・AIデータ等のCSVインポート
- ChatGPT（OpenAI）との連携による自然言語クエリ

**データの保存場所：** `C:\FARMS\{農場名}\farm.db`（農場ごとに独立したSQLiteデータベース）

---

## 2. 起動と農場選択

### 2-1. 起動

```
python main.py
```
または `restart_falcon.bat` をダブルクリックします。

農場選択ウィンドウが表示されます（`C:\FARMS` 配下のフォルダを自動検出）。

### 2-2. 農場を開く

1. リストから農場名を選択
2. 「**農場を開く**」ボタンをクリック（またはダブルクリック）

### 2-3. 新規農場を作成する

1. 「**新規農場作成**」ボタンをクリック
2. 農場名を入力して確定
3. `C:\FARMS\{農場名}` フォルダが作成され、空のDBが初期化される

### 3-1. コマンドの種類（タイプ選択）

| タイプ | 用途 |
|--------|------|
| **リスト** | 個体一覧を指定条件で抽出・表示 |
| **集計（経産牛）** | グループ別の頭数・平均値等を集計 |
| **イベントリスト** | 指定イベントの一覧を期間で抽出 |
| **イベント集計** | イベントをグループ・集計 |
| **受胎率（経産）** | 月別・産次別等の受胎率を計算 |
| **グラフ** | 散布図・折れ線等を描画 |
| **項目一括変更** | 複数個体の項目を一括更新 |

### 3-2. 補助入力フィールド

タイプによって表示される入力欄が変わります。

**リスト / 集計 共通：**
- **条件**：絞り込み条件（例：`DIM<150`、`LACT=1`、`RC=5`）
- **分類**：グループ化の軸（例：`産次`、`DIM30`）
- **内訳**：追加の集計軸
- **並び順**：ソートキー（例：`DIM`、`MILK desc`）

**イベントリスト：**
- イベント種別をチェックボックスで複数選択
- 開始日・終了日で絞り込み

**受胎率：**
- 受胎率の種類（月別 / 産次別 / SIRE別 / 授精師別等）
- 集計期間・条件

**グラフ：**
- グラフ種類（散布図 / 折れ線）
- Y軸項目・X軸項目・分類・条件

## 3-3. 条件の書き方（詳細）

「条件」欄に書いた条件に合う個体だけを絞り込みます。
空欄にすると全経産牛が対象になります。

---

### 基本①：比較演算子

`項目コード 演算子 数値` の形式で書きます。

| 演算子 | 意味 | 例 |
|--------|------|----|
| `<` | より小さい | `DIM<60` |
| `>` | より大きい | `DIM>200` |
| `<=` | 以下 | `DIM<=150` |
| `>=` | 以上 | `MILK>=30` |
| `=` | 等しい | `LACT=1` |
| `!=` | 等しくない | `LACT!=1` |

**よく使う例：**

| 条件 | 意味 |
|------|------|
| `DIM<60` | 分娩後60日未満 |
| `DIM>=100` | 分娩後100日以上 |
| `DIM>=60 DIM<=150` | DIM 60〜150日（後述の AND 複合条件） |
| `MILK>=35` | 直近乳量35kg以上 |
| `MILK<20` | 直近乳量20kg未満（低乳量個体） |
| `LACT=1` | 初産のみ |
| `LACT=2` | 2産のみ |
| `LACT>=3` | 3産以上 |
| `RC=2` | 繁殖コード＝分娩後未授精 |
| `RC=3` | 繁殖コード＝授精後 |
| `RC=5` | 繁殖コード＝妊娠 |
| `SCC>=50` | 体細胞数50万以上 |
| `LS>=4` | リニアスコア4以上（高SCC注意） |
| `BHB>=0.2` | BHB（ケトン体）0.2以上 |
| `BRED>=3` | 現産次の授精回数3回以上 |
| `DOPN>=150` | 空胎日数150日以上 |
| `DIMFAI>=80` | 初回授精DIM 80日以上（授精遅延） |

---

### 基本②：日本語コロン形式

`項目名：値` の形式でも書けます（全角コロン「：」でも半角「:」でも可）。

| 条件 | 意味 |
|------|------|
| `産次：初産` | 初産牛のみ（LACT=1 と同じ） |
| `産次：1産` | 初産牛のみ |
| `産次：経産` | 2産以上 |
| `産次：2産以上` | 2産以上 |
| `産次：2` | 2産のみ |
| `産次：3` | 3産のみ |

---

### 基本③：特殊キーワード

文字だけで書けるショートカットキーワードです。

| 条件 | 意味 |
|------|------|
| `受胎` | 妊娠中（RC=5）または乾乳中（RC=6）の個体 |
| `空胎` | 未妊娠（RC=1〜4）の個体 |

---

### 複合条件①：AND（すべての条件を満たす）

**スペースで区切って並べると AND（かつ）になります。**

```
DIM>=60 DIM<=150
```
→ DIM が 60日以上かつ150日以下（フレッシュ〜ピーク期）

```
LACT=1 DIM<150
```
→ 初産かつ DIM 150日未満

```
MILK>=30 LS>=4
```
→ 乳量30kg以上かつリニアスコア4以上（高乳量＋高SCC）

```
RC=2 DIM>=60
```
→ 分娩後未授精かつ DIM 60日以上（授精開始遅延の個体）

```
LACT>=3 MILK<20
```
→ 3産以上かつ乳量20kg未満（高産次の低産個体）

```
BRED>=3 RC=3
```
→ 授精回数3回以上かつ授精後（繰り返し不受胎の個体）

```
DOPN>=150 RC!=5
```
→ 空胎日数150日以上かつ未妊娠（長期空胎個体）

---

### 複合条件②：OR（いずれかの条件を満たす）

**`または` または `OR` で区切ると OR（または）になります。**

```
LACT=1 または LACT=2
```
→ 初産または2産

```
RC=2 または RC=4
```
→ 分娩後未授精または空胎

```
DIM<30 または DIM>300
```
→ 分娩直後（30日未満）または超長期（300日超）

---

### 複合条件③：AND と OR の組み合わせ

`または` の前後それぞれに AND 条件を並べます。

```
LACT=1 DIM<150 または LACT>=2 DIM<100
```
→ 「初産で DIM 150日未満」または「2産以上で DIM 100日未満」

```
RC=2 DIM>=60 または RC=4
```
→ 「分娩後未授精かつ60日超」または「空胎（繁殖不良疑い）」

---

### 条件の例：よくある用途別

**繁殖管理：**

| やりたいこと | 条件の書き方 |
|-------------|-------------|
| 授精をすべき個体（60日超未授精） | `RC=2 DIM>=60` |
| 繰り返し授精の個体 | `BRED>=3 RC!=5` |
| 妊娠鑑定予定（授精後35日前後） | `RC=3 DAI>=30 DAI<=45` |
| 長期空胎個体 | `DOPN>=150` |
| 初回授精が遅い個体 | `DIMFAI>=80` |

**乳量・健康管理：**

| やりたいこと | 条件の書き方 |
|-------------|-------------|
| 高SCC注意個体 | `LS>=4` |
| フレッシュ期の低乳量 | `DIM<60 MILK<20` |
| ケトーシスリスク | `BHB>=0.2 DIM<21` |
| 乾乳予定（高産次低乳量） | `LACT>=3 MILK<15` |

## 3-4. 分類の選び方（詳細）

「分類」欄はプルダウンから選びます。集計タイプで結果をグループに分けるときに使います。リスト表示でも列を追加する形で表示されます。

---

### 分類の選択肢と意味

| 分類 | 意味 | 使いどき |
|------|------|---------|
| **（空欄）** | 分類なし・全体を1行で集計 | 全体平均が知りたいとき |
| **産次** | 産次ごとに1行 | 産次1、産次2、産次3… と並ぶ |
| **産次（１産、２産、３産以上）** | 産次を3つにまとめる | 初産・2産・3産以上の比較 |
| **DIM7** | 7日ごとのグループ | 0-6日、7-13日、14-20日… |
| **DIM14** | 14日ごとのグループ | 0-13日、14-27日、28-41日… |
| **DIM21** | 21日ごとのグループ | 0-20日、21-41日、42-62日… |
| **DIM30** | 30日ごとのグループ | 0-29日、30-59日、60-89日… |
| **DIM50** | 50日ごとのグループ | 0-49日、50-99日、100-149日… |
| **DIM：産次** | DIM × 産次の2軸クロス集計 | 産次別・DIM帯別の両方を見たい |
| **空胎日数100日（DOPN=100）** | 空胎100日未満 / 100日以上の2グループ | 繁殖成績の振り分け |
| **空胎日数150日（DOPN=150）** | 空胎150日未満 / 150日以上の2グループ | 長期空胎個体の把握 |
| **分娩月** | 分娩した月ごとに集計 | 季節変動の分析 |

---

### カスタム分類（自由入力）

プルダウン以外に、直接入力でカスタム分類も使えます。

| 入力例 | 意味 |
|--------|------|
| `DIM=100` | DIM 100日未満 / 100日以上 の2グループ |
| `DIM=60,120` | DIM 60日未満 / 60-120日 / 120日以上 の3グループ |
| `DIM=60,120,200` | DIM を4グループに分割 |
| `DOPN=100` | 空胎100日未満 / 100日以上 の2グループ |
| `DOPN=100,150` | 空胎を3グループに分割 |

---

### 分類の使用例

**例1：産次別の乳量・SCC を比較したい**
```
タイプ: 集計（経産牛）
コマンド: MILK, SCC, LS
分類: 産次（１産、２産、３産以上）
```
→ 産次別に平均乳量・SCC・リニアスコアが一覧で出る

**例2：DIM 帯別の乳量を30日ごとに確認したい**
```
タイプ: 集計（経産牛）
コマンド: MILK, MFAT, MPROT
分類: DIM30
```
→ 0-29日、30-59日、60-89日…とグループ別に乳成分が出る

**例3：空胎150日以上の個体が産次別に何頭いるか確認したい**
```
タイプ: 集計（経産牛）
コマンド: MILK, DOPN
分類: 産次
条件: DOPN>=150
```
→ 長期空胎個体を産次別に集計

**例4：DIM × 産次のクロス集計で乳量の山を確認したい**
```
タイプ: 集計（経産牛）
コマンド: MILK
分類: DIM：産次
```
→ 産次1/2/3ごとに、DIM 帯別の平均乳量が出る（ラクテーションカーブを確認できる）

**例5：分娩月別に乳量の季節変動を見たい**
```
タイプ: 集計（経産牛）
コマンド: MILK, SCC
分類: 分娩月
```
→ 1月分娩・2月分娩…と月別に集計される

## 4. 個体カード

左メニューの「**🐄 個体カード**」から開きます。

### 4-1. 個体の検索・切り替え

- ウィンドウ上部の「個体ID:」入力欄にID（4桁）を入力してEnter
- 「◀」「▶」ボタンで前後の個体に移動

### 4-2. 表示内容

**左ペイン（個体カード）：**
- 管理番号（4桁）・JPN10（10桁個体識別番号）
- 産次、DIM（分娩後日数）、繁殖コード
- 乳量、乳脂率、乳蛋白率、SCC、リニアスコア
- 分娩予定日、直近AI日、授精回数 等

**右ペイン（イベント履歴）：**
- 登録されたイベントを時系列で一覧表示
- イベントをダブルクリックで詳細・編集

### 4-3. 除籍表示

売却（SOLD）・死亡（DEAD）の個体はヘッダーに赤い除籍バッジが表示されます。

## 5. イベント入力

左メニューの「**📝 イベント入力**」、または個体カードから開きます。

### 5-1. イベント登録の手順

1. 個体ID（4桁）を入力
2. イベント種別を選択（例：AI、CALV、MILK_TEST）
3. 日付を入力（デフォルトは今日）
4. イベント固有の項目を入力
5. 「登録」ボタンをクリック

### 5-2. 主なイベントの入力項目

| イベント | 主な入力項目 |
|----------|-------------|
| **AI**（人工授精） | SIRE（精液番号）、授精師、授精種類（AI/ET） |
| **CALV**（分娩） | 難産レベル（1-5）、子牛性別・ID・JPN10 |
| **MILK_TEST**（乳検） | 乳量、乳脂率、乳蛋白率、SCC、リニアスコア、BHB等 |
| **PDP**（妊娠プラス） | 受胎AI/ETの特定 |
| **PDN**（妊娠マイナス） | 子宮・卵巣所見 |
| **SOLD**（売却） | 売却先 |
| **DEAD**（死亡） | 死因 |
| **MAST**（乳房炎） | 罹患乳区、起因菌 |

## 6. 繁殖検診表

左メニューの「**🩺 繁殖検診**」から開きます。

### 6-1. 機能概要

検診予定日ごとに対象個体を抽出し、一覧で確認・印刷できます。

### 6-2. 操作手順

1. **検診予定日**をプルダウンから選択
2. 「**抽出**」ボタンをクリック → 対象個体が一覧表示される
3. 不要な個体のチェックを外す（全選択/全解除ボタンあり）
4. 「**印刷プレビュー**」で確認してから印刷

### 6-3. 表示列のカスタマイズ

「**カスタマイズ**」ボタンで表示する項目（列）を項目辞書から追加/削除できます。設定は農場ごとに保存されます。

## 7. 分析・レポート

左メニューの「**📋 レポート作成**」またはメイン画面のコマンド入力から実行します。

### 7-1. リスト表示

個体の一覧を任意の条件と項目で表示します。

**例：**
```
タイプ: リスト
コマンド: LACT, DIM, MILK, RC
条件: DIM<150
並び順: DIM
```
→ DIM150日以内の個体を産次・DIM・乳量・繁殖コード付きで表示

### 7-2. 集計

グループ別に頭数・平均値等を集計します。

**例：**
```
タイプ: 集計（経産牛）
コマンド: MILK, DIM
分類: LACT
```
→ 産次別の平均乳量・平均DIMを集計

### 7-3. 受胎率

```
タイプ: 受胎率（経産）
受胎率の種類: 月別
```
→ 月別の授精頭数・受胎頭数・受胎率を表示

### 7-4. グラフ

```
タイプ: グラフ
グラフ種類: 散布図
X軸: DIM
Y軸: MILK
分類: LACT
```
→ 産次別に色分けした DIM × 乳量の散布図

### 7-5. 結果の出力

結果テーブルを右クリックまたはツールバーから：
- **Excelで開く**（.csv出力）
- **印刷**
- **クリップボードにコピー**

## 9. データ吸い込み（インポート）

左メニューの「**📥 データ吸い込み**」から各インポート画面を開きます。

### 9-1. 対応フォーマット

| 種別 | 説明 |
|------|------|
| **乳検データ** | DHI/農協乳検CSVを取り込み、MILK_TESTイベントを一括登録 |
| **DC305** | 搾乳ロボット・管理ソフトDC305のデータをインポート |
| **AI/ETデータ** | 授精記録CSVをAI/ETイベントとして登録 |
| **ゲノム検査** | ゲノム解析結果をGENOMICイベントとして登録 |
| **個体登録** | 牛の基本情報（JPN10、生年月日、品種等）を一括登録 |

### 9-2. インポートの一般手順

1. インポート種別を選択

4. プレビューで内容を確認
