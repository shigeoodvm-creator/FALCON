## 1. アプリ概要

**FALCON2** は酪農家向けの牛群管理システムです。

- 個体ごとの繁殖・健康・乳量データをイベントとして記録
- 受胎率・乳量・リニアスコア等をリスト/集計/グラフで分析
- 繁殖検診表の作成・印刷
- 乳検データ・DC305・AIデータ等のCSVインポート
- ChatGPT（OpenAI）との連携による自然言語クエリ

**データの保存場所：** `C:\FARMS\{農場名}\farm.db`（農場ごとに独立したSQLiteデータベース）

---

## 2. 起動と農場選択

### 2-1. 起動

```
python main.py
```
または `restart_falcon.bat` をダブルクリックします。

農場選択ウィンドウが表示されます（`C:\FARMS` 配下のフォルダを自動検出）。

### 2-2. 農場を開く

1. リストから農場名を選択
2. 「**農場を開く**」ボタンをクリック（またはダブルクリック）

### 2-3. 新規農場を作成する

1. 「**新規農場作成**」ボタンをクリック
2. 農場名を入力して確定
3. `C:\FARMS\{農場名}` フォルダが作成され、空のDBが初期化される

```

│  🩺 繁殖検診      │  │ タイプ: [リスト ▼]           │  │
│  📝 イベント入力  │  │ コマンド: [___________] 実行  │  │
│  📋 レポート作成  │  └─────────────────────────────┘  │
│  📊 ダッシュボード│                                   │
│  📥 データ吸い込み│  最終分娩: — 最終AI: — 最終乳検: — │

└─────────────────────────────────────────────────────┘
```

### 3-1. コマンドの種類（タイプ選択）

| タイプ | 用途 |
|--------|------|
| **リスト** | 個体一覧を指定条件で抽出・表示 |
| **集計（経産牛）** | グループ別の頭数・平均値等を集計 |
| **イベントリスト** | 指定イベントの一覧を期間で抽出 |
| **イベント集計** | イベントをグループ・集計 |
| **受胎率（経産）** | 月別・産次別等の受胎率を計算 |
| **グラフ** | 散布図・折れ線等を描画 |
| **項目一括変更** | 複数個体の項目を一括更新 |

### 3-2. 補助入力フィールド

タイプによって表示される入力欄が変わります。

**リスト / 集計 共通：**
- **条件**：絞り込み条件（例：`DIM<150`、`LACT=1`、`RC=5`）
- **分類**：グループ化の軸（例：`LACT`、`PEN`）
- **内訳**：追加の集計軸
- **並び順**：ソートキー（例：`DIM`、`MILK desc`）

**イベントリスト：**
- イベント種別をチェックボックスで複数選択
- 開始日・終了日で絞り込み

**受胎率：**
- 受胎率の種類（月別 / 産次別 / SIRE別 / 授精師別等）
- 集計期間・条件

**グラフ：**
- グラフ種類（散布図 / 折れ線）
- Y軸項目・X軸項目・分類・条件

## 4. 個体カード

左メニューの「**🐄 個体カード**」から開きます。

### 4-1. 個体の検索・切り替え

- ウィンドウ上部の「個体ID:」入力欄にID（4桁）を入力してEnter
- 「◀」「▶」ボタンで前後の個体に移動

### 4-2. 表示内容

**左ペイン（個体カード）：**
- 管理番号（4桁）・JPN10（10桁個体識別番号）
- 産次、DIM（分娩後日数）、繁殖コード
- 乳量、乳脂率、乳蛋白率、SCC、リニアスコア
- 分娩予定日、直近AI日、授精回数 等

**右ペイン（イベント履歴）：**
- 登録されたイベントを時系列で一覧表示
- イベントをダブルクリックで詳細・編集

### 4-3. 除籍表示

売却（SOLD）・死亡（DEAD）の個体はヘッダーに赤い除籍バッジが表示されます。

## 5. イベント入力

左メニューの「**📝 イベント入力**」、または個体カードから開きます。

### 5-1. イベント登録の手順

1. 個体ID（4桁）を入力
2. イベント種別を選択（例：AI、CALV、MILK_TEST）
3. 日付を入力（デフォルトは今日）
4. イベント固有の項目を入力
5. 「登録」ボタンをクリック

### 5-2. 主なイベントの入力項目

| イベント | 主な入力項目 |
|----------|-------------|
| **AI**（人工授精） | SIRE（精液番号）、授精師、授精種類（AI/ET） |
| **CALV**（分娩） | 難産レベル（1-5）、子牛性別・ID・JPN10 |
| **MILK_TEST**（乳検） | 乳量、乳脂率、乳蛋白率、SCC、リニアスコア、BHB等 |
| **PDP**（妊娠プラス） | 受胎AI/ETの特定 |
| **PDN**（妊娠マイナス） | 子宮・卵巣所見 |
| **SOLD**（売却） | 売却先 |
| **DEAD**（死亡） | 死因 |
| **MAST**（乳房炎） | 罹患乳区、起因菌 |

## 6. 繁殖検診表

左メニューの「**🩺 繁殖検診**」から開きます。

### 6-1. 機能概要

検診予定日ごとに対象個体を抽出し、一覧で確認・印刷できます。

### 6-2. 操作手順

1. **検診予定日**をプルダウンから選択
2. 「**抽出**」ボタンをクリック → 対象個体が一覧表示される
3. 不要な個体のチェックを外す（全選択/全解除ボタンあり）
4. 「**印刷プレビュー**」で確認してから印刷

### 6-3. 表示列のカスタマイズ

「**カスタマイズ**」ボタンで表示する項目（列）を項目辞書から追加/削除できます。設定は農場ごとに保存されます。

---

## 7. 分析・レポート

左メニューの「**📋 レポート作成**」またはメイン画面のコマンド入力から実行します。

### 7-1. リスト表示

個体の一覧を任意の条件と項目で表示します。

**例：**
```
タイプ: リスト
コマンド: LACT, DIM, MILK, RC
条件: DIM<150
並び順: DIM
```
→ DIM150日以内の個体を産次・DIM・乳量・繁殖コード付きで表示

### 7-2. 集計

グループ別に頭数・平均値等を集計します。

**例：**
```
タイプ: 集計（経産牛）
コマンド: MILK, DIM
分類: LACT
```
→ 産次別の平均乳量・平均DIMを集計

### 7-3. 受胎率

```
タイプ: 受胎率（経産）
受胎率の種類: 月別
```
→ 月別の授精頭数・受胎頭数・受胎率を表示

### 7-4. グラフ

```
タイプ: グラフ
グラフ種類: 散布図
X軸: DIM
Y軸: MILK
分類: LACT
```
→ 産次別に色分けした DIM × 乳量の散布図

### 7-5. 結果の出力

結果テーブルを右クリックまたはツールバーから：
- **Excelで開く**（.csv出力）
- **印刷**
- **クリップボードにコピー**

## 8. ダッシュボード

左メニューの「**📊 ダッシュボード**」から開きます。

ブラウザ形式のHTMLレポートが表示されます。

### 表示内容

| セクション | 内容 |
|-----------|------|
| **頭数サマリー** | 経産・未経産・乾乳・除籍の内訳 |
| **産次分布** | 産次別のドーナツチャート（1産 / 2産 / 3産以上） |
| **乳量統計** | 平均・最大・最小・標準偏差 |
| **リニアスコア分布** | SCC高リスク個体のハイライト |
| **散布図** | DIM × 乳量 の分布 |

## 9. データ吸い込み（インポート）

左メニューの「**📥 データ吸い込み**」から各インポート画面を開きます。

### 9-1. 対応フォーマット

| 種別 | 説明 |
|------|------|
| **乳検データ** | DHI/農協乳検CSVを取り込み、MILK_TESTイベントを一括登録 |
| **DC305** | 搾乳ロボット・管理ソフトDC305のデータをインポート |
| **AI/ETデータ** | 授精記録CSVをAI/ETイベントとして登録 |
| **ゲノム検査** | ゲノム解析結果をGENOMICイベントとして登録 |
| **個体登録** | 牛の基本情報（JPN10、生年月日、品種等）を一括登録 |

### 9-2. インポートの一般手順

1. インポート種別を選択
2. CSVファイルを指定（ファイル選択ダイアログ）
3. 列のマッピングを確認・修正
4. プレビューで内容を確認
5. 「**取り込み実行**」ボタンをクリック

---

## 10. 辞書・設定

左メニューの「**📚 辞書・設定**」から各設定画面を開きます。

### 10-1. 項目辞書

表示・集計・カード表示に使う「項目」の定義を管理します。

- **表示名・別名の変更**：`MILK` を「乳量」として表示するなど
- **計算式の編集**：カスタム計算項目（`calc`型）の定義
- **新規項目の追加**：独自の管理項目を追加

### 10-2. イベント辞書

登録できるイベントの種別・入力項目を管理します。

### 10-3. 授精設定

- 授精師の登録・管理
- 授精種類（AI / ET / 自然交配等）の設定

### 10-4. ペン（群）設定

牛を管理する「群」の名称・順序を設定します。

### 10-5. 薬剤設定

繁殖処置・治療で使用する薬剤リストを管理します。

### 10-6. 農場設定

農場名、基本情報などを編集します。

### 10-7. アプリ設定

バックアップ設定、OpenAI APIキー等の全体設定を管理します。

## 11. ChatGPT連携

メイン画面右上のチャット入力エリアから自然言語でクエリを入力できます。

**例：**
```
直近30日以内にAIした牛のリストを出して
```
```
産次1で空胎日数が長い牛トップ10を教えて
```

### 設定

`config/openai.json` にOpenAI APIキーを設定する必要があります：

```json
{
  "api_key": "sk-xxxx..."
}
```

または「**⚙️ アプリ設定**」→「OpenAI設定」から入力できます。

## 13. 主要項目一覧

分析コマンドで使用できる主な項目コードです。

### 基本情報

| コード | 説明 |
|--------|------|
| `COW_ID` | 管理番号（4桁） |
| `JPN10` | 個体識別番号（10桁） |
| `LACT` | 産次 |
| `BRD` | 品種 |

**繁殖コード（RC）の値：**

| 値 | 意味 |
|----|------|
| 1 | 繁殖停止 |
| 2 | 分娩後（未授精） |
| 3 | 授精後 |
| 4 | 空胎 |
| 5 | 妊娠 |
| 6 | 乾乳 |

### 繁殖関連

| コード | 説明 |
|--------|------|
| `LASTAI` | 最終AI日 |
| `DUE` | 分娩予定日 |
| `BRED` | 現産次の授精回数 |
| `LSIR` | 最終使用SIRE |
| `CSIR` | 受胎SIRE |
| `DOPN` | 空胎日数（現産次） |
| `DIMFAI` | 初回授精DIM |
| `PGEST` | 前産次在胎日数 |

### 乳検（DHI）関連

| コード | 説明 |
|--------|------|
| `MILK` | 直近乳量（kg） |
| `MFAT` | 乳脂率（%） |
| `MPROT` | 乳蛋白率（%） |
| `SCC` | 体細胞数（万/mL） |
| `LS` | リニアスコア |
| `MUN` | MUN（乳中尿素窒素） |
| `BHB` | BHB（ケトン体） |
| `TDATE` | 直近乳検日 |
| `TDIM` | 直近乳検日のDIM |
| `PMILK` | 前回乳量 |
| `PSCC` | 前回SCC |

## 付録：よくある操作

### 今日AI（授精）した牛を確認したい

```
タイプ: イベントリスト
イベント: AI
開始日: 今日 / 終了日: 今日
```

### 受胎率を月別に確認したい

```
タイプ: 受胎率（経産）
受胎率の種類: 月別
```

### DIM 100日以上で未受胎の牛を探したい

```
タイプ: リスト
コマンド: COW_ID, LACT, DIM, LASTAI, RC
条件: DIM>=100 AND RC=2
並び順: DIM desc
```

### 産次別の平均乳量を比べたい

```
タイプ: 集計（経産牛）
コマンド: MILK, DIM, SCC
分類: LACT
```

### ゲノム検査結果を確認したい

左メニュー「**📋 レポート作成**」→「**ゲノムレポート**」から開きます。
