# Phase E-2: コマンド入力UI最終整理 実装サマリー

## 概要

FALCON2のコマンド入力UIを最終整理し、初見ユーザーでも「何ができるか」「今どのモードか」が一瞬で分かる状態にしました。

**実装ファイル**: 
- `app/ui/main_window.py`（修正）

## 実装内容

### 1. コマンド入力欄のプレースホルダー改善

**仕様**:
- 未入力時のみ表示
- 入力開始と同時に消える
- 説明文ではなく「実際に使える例」に限定

**表示例**:
```
例）101 または レポート：月別分娩頭数 2024年 CSV または 分析：最近6か月で2産の受胎率が下がった理由
```

**実装**:
- `_show_command_placeholder()`: プレースホルダーを表示
- `_hide_command_placeholder()`: プレースホルダーを非表示
- `_on_command_key_press()`: キー入力時にプレースホルダーを非表示
- `_on_command_focus_in()`: フォーカス取得時にプレースホルダーを非表示
- `_on_command_focus_out()`: フォーカス喪失時にプレースホルダーを表示（空の場合）

### 2. モード状態の視覚的明確化

**表示位置**: コマンド入力欄の上

**表示例**:
- **通常モード**（グレー）
- **分析モード（DB限定）**（ブルー）
- **定型レポート実行中**（グリーン）

**実装**:
- `mode_label`: モード状態を表示するラベル
- `_update_mode_label(text, color)`: モード状態ラベルを更新
- 各モード切り替え時に自動更新

**色分け**:
- 通常モード: `gray`
- 分析モード: `blue`
- 定型レポート: `green`

### 3. 入力補助（軽量・安全）

**仕様**:
- 入力が「レポート：」「報告：」で始まった場合のみ、定型レポート名の候補をドロップダウン表示
- 自動確定しない（選択式）
- Enterで通常どおり実行可能
- 通常モード入力（数値など）には一切影響しない

**候補内容**:
- 月別分娩頭数
- 月別授精頭数
- 月別受胎率
- 2産のみ分娩
- DIM分布

**実装**:
- `_show_report_dropdown(current_text)`: ドロップダウンを表示
- `_update_report_dropdown(current_text)`: ドロップダウンの候補を更新
- `_hide_report_dropdown()`: ドロップダウンを非表示
- `_on_report_dropdown_select(event)`: ドロップダウンから選択された時
- 入力テキストに応じて候補をフィルタリング

### 4. 初回起動時の最小ガイド（1回のみ）

**仕様**:
- 初回起動時のみ、messageboxで表示
- 設定ファイル（`farm_settings.json`）で「表示済み」を保存
- 次回以降は表示しない

**表示内容**:
```
FALCON2 クイックガイド

【基本的な使い方】

• 数字を入力 → 個体カードを開く
  例：101

• 「レポート：」→ 定型集計
  例：レポート：月別分娩頭数 2024年 CSV

• 「分析：」→ DB限定の分析モード
  例：分析：最近6か月で2産の受胎率が下がった理由

• CSV / Excel を付けると出力されます
  例：レポート：月別分娩頭数 Excel

【ヒント】
• 「定型レポート一覧」ボタンで利用可能なレポートを確認できます
• コマンド入力欄の例示テキストも参考にしてください
```

**実装**:
- `_show_quick_guide_if_first_time()`: 初回起動時のガイドを表示
- `SettingsManager`を使用して`quick_guide_shown`フラグを保存
- `farm_settings.json`に保存

## 使用例

### 例1: プレースホルダーの表示

1. コマンド入力欄が空の状態でフォーカスが外れる
2. プレースホルダーが表示される
3. 入力開始と同時に消える

### 例2: モード状態の表示

1. 通常モード: 「通常モード」（グレー）が表示される
2. 分析モード: 「分析モード（DB限定）」（ブルー）が表示される
3. 定型レポート: 「定型レポート実行中」（グリーン）が表示される

### 例3: 入力補助の使用

1. コマンド入力欄に「レポート：」と入力
2. ドロップダウンが表示され、定型レポート名の候補が表示される
3. 候補をダブルクリックまたはEnterで選択
4. 選択されたレポート名が入力欄に挿入される

### 例4: 初回ガイドの表示

1. 初回起動時のみ、messageboxでガイドが表示される
2. 次回以降は表示されない（`farm_settings.json`に保存）

## 技術的詳細

### プレースホルダー実装

- Entryウィジェットの`foreground`属性を使用してグレー表示
- フォーカス状態を管理して表示/非表示を切り替え
- 入力開始時に即座に非表示

### モード状態表示

- Labelウィジェットを使用
- 色分けで視覚的に区別
- 各モード切り替え時に自動更新

### 入力補助ドロップダウン

- Listboxウィジェットを使用
- 入力テキストに応じて候補をフィルタリング
- ダブルクリックまたはEnterで選択
- 選択後は自動的に非表示

### 初回ガイド

- `SettingsManager`を使用して設定を保存
- `farm_settings.json`の`quick_guide_shown`フラグで管理
- 初回のみ表示、次回以降は表示しない
