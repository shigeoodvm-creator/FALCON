## 動作確認項目

### 1. 分析UIの起動確認
- [ ] メイン画面の「分析」ボタンからAnalysisUIが表示される
- [ ] 既存の画面遷移や他メニューに影響が出ていないか

### 2. 基本動線の確認（E2E）
- [ ] 右パネルから項目をクリック／ダブルクリックして入力できるか

- [ ] コマンド入力 → 実行 → 結果表示（表／グラフ）が一連で動作するか

### 3. 各分析種別の最低限テスト
- [ ] リスト：項目＋条件で結果が出るか
- [ ] 集計：GROUP BYが効いているか
- [ ] イベント集計：イベント指定が正しく解釈されるか
- [ ] グラフ：種類指定（折れ線・棒など）が反映されるか
- [ ] 繁殖分析：受胎率などの指標がエラーなく計算されるか（仮実装でも可）

### 4. エラーハンドリング
- [ ] 未解決の項目名／イベント名を入力した場合に、分かりやすいエラーや警告が出るか
- [ ] エラー時にUIが壊れず、再入力できるか
