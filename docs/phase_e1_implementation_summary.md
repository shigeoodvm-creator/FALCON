# Phase E-1: 定型レポート機能 実装サマリー

## 概要

分析モードのSQLテンプレートを再利用し、「定型レポート」を日本語1行で呼び出せる仕組みを実装しました。

**実装ファイル**: 
- `app/modules/analysis_reports.py`（新規）
- `app/ui/main_window.py`（修正）

## 実装内容

### 1. 定型レポート定義ファイル

**ファイル**: `app/modules/analysis_reports.py`

**クラス**: `AnalysisReports`

**内容**:
- 辞書形式で定型レポートを定義
- 各レポートは以下を持つ：
  - `display_name`: 日本語表示名
  - `description`: 簡潔な説明（UI表示用）
  - `template_name`: `analysis_sql_templates` のテンプレート名
  - `default_params`: デフォルトパラメータ（期間など）
  - `default_output`: デフォルト出力形式（`["screen"]`, `["csv"]`, `["excel"]`, `["csv","excel"]`）

### 2. 実装済み定型レポート（5つ）

#### calving_month_lact_report
- **表示名**: 月別分娩頭数（産次別）
- **使用テンプレート**: `calving_by_month_lact`
- **デフォルト期間**: 直近12か月
- **デフォルト出力**: 画面表示

#### insemination_month_lact_report
- **表示名**: 月別授精頭数（産次別）
- **使用テンプレート**: `insemination_by_month_lact`
- **デフォルト期間**: 直近12か月
- **デフォルト出力**: 画面表示

#### conception_rate_month_lact_report
- **表示名**: 月別受胎率（産次別）
- **使用テンプレート**: `conception_rate_by_month_lact`
- **デフォルト期間**: 直近12か月
- **デフォルト出力**: 画面表示

#### lact2_calving_report
- **表示名**: 2産のみ分娩頭数
- **使用テンプレート**: `calving_lact2_only`
- **デフォルト期間**: 直近12か月
- **デフォルト出力**: 画面表示

#### dim_distribution_report
- **表示名**: DIM分布
- **使用テンプレート**: `dim_distribution`
- **デフォルト期間**: 直近12か月
- **デフォルト出力**: 画面表示

### 3. レポート呼び出し方法

以下のいずれかで定型レポートを実行可能：

#### 入力文の接頭辞
- 「レポート：」で始まる入力
- 「報告：」で始まる入力

**例**:
- `レポート：月別分娩頭数`
- `レポート：2産のみ分娩`
- `報告：月別受胎率`

**動作**:
- `analysis_mode` を自動的に有効化
- 対応する `report_key` を解決
- 内部で SQLテンプレート + パラメータを展開
- Phase D と同じ分析実行フローへ渡す

### 4. 期間指定ルール

#### 期間指定が無い場合
- `report.default_params` に従う
- 例：直近12か月

#### 日本語で期間が指定された場合
以下の形式に対応：

- **「2024年」形式**: その年の1月1日～12月31日
- **「直近Nか月」形式**: 直近Nか月（例：「直近6か月」）
- **「N〜M月」形式**: 今年のN月～M月（例：「3〜8月」）

**例**:
- `レポート：月別分娩頭数 2024年`
- `レポート：月別分娩頭数 直近6か月`
- `レポート：月別分娩頭数 3〜8月`

### 5. 出力形式

#### デフォルト出力
- `report.default_output` に従う
- 通常は `["screen"]`（画面表示のみ）

#### ユーザー指定で上書き
入力文に以下が含まれていれば上書き：
- 「CSV」→ CSV出力
- 「Excel」→ Excel出力
- 「CSV+Excel」→ 両方出力

**例**:
- `レポート：月別分娩頭数 CSV`
- `レポート：月別分娩頭数 Excel`
- `レポート：月別分娩頭数 CSV+Excel`

### 6. UI表示（簡易）

#### 定型レポート一覧ボタン
- 分析モードUI内に「定型レポート一覧」ボタンを追加
- 押すと以下を表示：
  - 表示名
  - 簡単な説明
  - 使用方法

**実行方法**:
- コマンドラインから実行（UIボタンは一覧表示のみ）

## 技術的詳細

### 期間計算ロジック

- `last_12_months`: 直近12か月（365日前～今日）
- `this_year`: 今年（1月1日～今日）
- `custom`: カスタム期間（開始日・終了日を指定）

### 期間パーサ

- 正規表現を使用して日本語テキストから期間を抽出
- 対応形式：
  - `(\d{4})年`: 年指定
  - `直近(\d+)か月`: 直近Nか月
  - `(\d+)〜(\d+)月`: 月範囲（今年を前提）

### レポート解決ロジック

1. 入力文から「レポート：」「報告：」を検出
2. 表示名を抽出
3. 表示名からレポートキーを検索（部分一致）
4. レポート定義を取得
5. SQLテンプレートを展開
6. 分析モードフローへ渡す
