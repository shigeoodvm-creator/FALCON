Problem

「分析したいのに、できない」
その壁を取り除きます

DairyComp305 は世界標準の牛群管理ソフトですが、
日本の現場には大きな3つの障壁がありました。

🌐

英語 ＋ コマンド操作

DC305 はすべての操作がコマンドライン入力。英語の専門用語を習得しなければ 基本的な集計すらできず、現場の獣医師には敷居が高すぎました。

💰

高額な導入コスト

海外製ソフトは本体費用に加えて保守・サポート費用も高額。 個人で担当農場に導入を勧めるには、費用面での説明が難しい状況でした。

📊

分析結果の農場還元

深い分析ができても、結果をわかりやすく農家に伝えるには 別途レポート作成の手間がかかります。多忙な現場では後回しになりがちでした。

Solution

FALCON が選ばれる4つの理由

現場の獣医師が設計したから、現場で本当に使えます。

1

完全日本語 ＋ GUI操作

コマンド不要。すべての操作をマウスで完結できます。 乳検データ・ゲノムレポートの取り込みも数クリック。 初めての獣医師でも、導入当日から使えます。

2

DairyComp305互換の分析エンジン

21日妊娠率（21-day Pregnancy Rate）をはじめ、 DC305準拠のロジックで牛群の繁殖成績を正確に算出。 世界標準指標で農場をベンチマーク評価できます。

3

ダッシュボードによる自動分析

繁殖・分娩・乳量・ゲノムの主要指標をダッシュボードに自動集計。 複雑な分析が得意でない獣医師でも、訪問前に農場の状態を 一目で把握し、一定水準以上の情報を農家に還元できます。

4

現場獣医師による継続開発

NOSAI北海道で20年以上の臨床経験を持つ獣医師が、 実際の担当農場を通じて仕様を磨き続けています。 「現場で本当に必要な機能」だけを実装した実践的なソフトです。

Features

主な機能

繁殖から乳質・ゲノムまで、酪農経営に必要な分析を網羅します。

繁殖管理

21日妊娠率（21-day PR）

DairyComp305準拠ロジックで繁殖成績を算出。サイクル別・月別の推移グラフで問題サイクルを即座に特定。

繁殖管理

繁殖検診サポート

検診結果の入力・集計から、受胎率・妊娠率・空胎日数の追跡まで一元管理。

個体管理

個体カード

1頭ごとの繁殖・分娩・治療・乳量・ゲノム情報を時系列で可視化。過去のイベントも即検索。

アラート

長期空胎・分娩超過チェック

300日超空胎牛・分娩予定15日超過牛を自動抽出。見落としをゼロに。

データ連携

乳検・ゲノムデータ取込

DHI乳検データ・ゲノム評価レポートをワンクリックで取り込み。既存データを無駄にしません。

データ連携

DairyComp305 データ移行

DC305形式のデータをFALCONに取り込み可能。既存のDC305ユーザーも乗り換えできます。

ダッシュボード

農場サマリー自動生成

訪問前にダッシュボードを確認するだけで、その日の農場状況を把握。農家への説明資料も自動作成。

複数農場

マルチファーム対応

担当農場を1台のPCで管理。農場間の切り替えも瞬時に。担当が増えても追加費用なし。

オフライン

インターネット不要

農場でのネット環境に依存せず動作。データはローカルに保存されるため、通信のない牛舎でも安心。

Case Study

データ分析が、農場を変える。

FALCONの設計思想は、開発者自身の臨床経験から生まれています。

CASE REPORT — 北獣会誌 59巻（2015年）

「妊娠率に着目した繁殖管理分析により成績が大幅改善した1事例」

経産牛80頭のフリーストール農場。2012年度の平均妊娠率はわずか4%、 受胎率も8%と著しく低値だった。個体として繁殖障害に摘発されるものは少なく、 原因の特定が困難な状態だった。

DairyComp305を用いて21日サイクルごとの妊娠率・発情発見率・受胎率を分析。 発情発見率は問題なく、受胎率が通年で著しく低いこと、 そして季節性がないことを確認。「人工授精手技の問題」という根本原因を特定し、 AI外部委託という対策を提案・実施した。

妊娠頭数は20頭→46頭へ増加。農場は経済的苦境を脱し、 出荷乳量の大幅増加・導入頭数ゼロという成果を達成した。
「獣医師の知識に基づく繁殖管理情報の分析は、問題点の顕在化と解決策立案に有効である」 ──これがFALCONの根本設計思想です。

現場の獣医師が、
現場のために作った。

FALCONは、NOSAI北海道に勤務する獣医師・大脇茂雄が、 20年以上の酪農臨床経験を通じて開発した牛群管理ソフトです。

繁殖・消化器外科・フレッシュカウ管理・蹄病・乳質と幅広い臨床研究に取り組み、 2010年には牛出血性腸症候群の病理学的研究で 日本獣医師会獣医学術賞（産業動物部門・全国大会）を受賞。 農林水産省経営局長賞も2度（2008年・2017年）獲得し、 北海道獣医師会誌 優秀論文賞（2015年）、 産業動物獣医学会北海道地区学会長賞（2023年）と 地域から全国まで一貫して評価されてきた実績を持ちます。 2023年には海外製牛群管理ソフトの解説論文を 家畜診療誌に発表するなど、 日本の酪農現場への知識普及にも積極的に取り組んでいます。

「DC305に匹敵する分析を、英語が得意でない獣医師でも 使えるようにしたい」──その思いがFALCON開発の出発点です。
