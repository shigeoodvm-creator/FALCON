## 次に着手する改善

### 選択: C3. 繁殖分析の完全実装

**理由**:
1. **獣医師の仮説検証力に直結**: 繁殖指標（受胎率、発情発見率、妊娠率など）は、獣医師が最も必要とする分析指標
2. **即座の価値提供**: 繁殖成績の定量的評価により、意思決定をデータドリブンに行える
3. **既存インフラの活用**: FormulaEngineやRuleEngineのロジックを活用できるため、実装効率が高い
4. **v1.0の自然な拡張**: 既にREPRO分析種別は実装済みで、計算ロジックの実装のみで完了可能

**実装方針**:
- ExecutorV2の`_execute_repro()`を完全実装
- 繁殖指標の計算ロジックを実装
  - 受胎率: 授精回数に対する妊娠確定の割合
  - 発情発見率: 空胎期間中の発情発見の割合
  - 妊娠率: 授精後一定期間内の妊娠確定の割合
  - 空胎日数: 分娩後から妊娠確定までの日数
  - 初回授精日数: 分娩後から初回授精までの日数
- QueryRouterV2で繁殖指標の解析を追加
- 結果表示を表形式で実装
