# FALCON2 配布前チェックリスト

## 目的
EXE配布・現場運用前提での最終確認項目

---

## 【E-3.1】データなし／未設定農場の安全確認

### 起動確認
- [ ] アプリが正常に起動する
- [ ] 農場選択画面が表示される
- [ ] 農場を選択してメイン画面が開く

### 農場未選択時の挙動
- [ ] 農場を選択せずに閉じた場合、エラーが出ない
- [ ] 農場が見つからない場合、適切なメッセージが表示される

### cow テーブルが 0 件の場合
- [ ] アプリが起動できる
- [ ] 個体カードを開こうとしてもエラーが出ない
- [ ] 「該当データがありません」などの適切なメッセージが表示される

### event テーブルが 0 件の場合
- [ ] 定型レポート実行が落ちない
  - [ ] 月別分娩頭数（産次別）
  - [ ] 月別授精頭数（産次別）
  - [ ] 月別受胎率（産次別）
  - [ ] 2産のみ分娩頭数
  - [ ] DIM分布
- [ ] 分析モード実行が落ちない
- [ ] CSV / Excel 出力が落ちない（ヘッダー行のみ出力される）

### 結果が 0 件の場合の統一挙動
- [ ] SQL結果が 0 行の場合、画面表示で「該当データがありません」と表示される
- [ ] CSV出力時、ヘッダー行のみ出力される（0行データ）
- [ ] Excel出力時、ヘッダー行のみ出力される（0行データ）
- [ ] 例外や traceback が表示されない

## 【E-3.2】パス・権限・EXE化前提チェック

### 出力ディレクトリの自動作成
- [ ] `farms/<farm_name>/exports/analysis/` が存在しない場合、自動作成される
- [ ] ディレクトリ作成に失敗した場合、適切なエラーメッセージが表示される

### パス設計の統一
- [ ] 相対パスのみ使用されている（絶対パス禁止）
- [ ] カレントディレクトリ依存が排除されている
- [ ] Windows EXE 実行時でも動作する

### ファイル書き込み失敗時の処理
- [ ] 権限エラー等が発生した場合、messagebox で簡潔に通知される
- [ ] アプリは落とさない
- [ ] エラーメッセージが業務向けで分かりやすい

---

## 【E-3.3】エラー表示の最終整理

### trace / stacktrace をユーザーに見せない
- [ ] Exception は内部ログにのみ出力される
- [ ] UI には業務向け文言のみ表示される
- [ ] コンソールに traceback が表示されない（EXE実行時）

### messagebox 文言の統一
- [ ] タイトルが「処理を実行できませんでした」で統一されている
- [ ] データなしの場合：「対象となるデータがありませんでした」
- [ ] 出力失敗の場合：「ファイルの書き込みに失敗しました。フォルダの権限を確認してください」
- [ ] 想定外の場合：「予期しないエラーが発生しました。操作をやり直してください」

### 通常モード／分析モード共通で文言を揃える
- [ ] エラーメッセージが統一されている
- [ ] 0件の場合のメッセージが統一されている

## 【機能確認】

### 通常モード確認
- [ ] 数字を入力 → 個体カードを開く
- [ ] 存在しない牛IDを入力 → エラーが出ない
- [ ] 数値以外を入力 → 何も起こらない（エラー表示なし）

### 分析モード確認
- [ ] 「分析：」で始まる入力で分析モードが有効化される
- [ ] 「解説：」「理由：」でも分析モードが有効化される
- [ ] 分析モードチェックボックスで有効化できる
- [ ] SQL実行結果が正しく表示される
- [ ] 0件の場合、「該当データがありません」と表示される

### 定型レポート確認
- [ ] 「レポート：月別分娩頭数」で実行できる
- [ ] 「報告：月別分娩頭数」でも実行できる
- [ ] 期間指定（「2024年」「直近6か月」「3〜8月」）が動作する
- [ ] 0件の場合でもエラーが出ない

### CSV / Excel 出力確認
- [ ] CSV出力チェックボックスで出力できる
- [ ] Excel出力チェックボックスで出力できる
- [ ] 入力文に「CSV」「Excel」を含めても出力できる
- [ ] 0件の場合、ヘッダー行のみ出力される
- [ ] 出力ファイルが正しい場所に保存される
- [ ] 権限エラー時、適切なメッセージが表示される

---

## 【UI確認】

### コマンド入力欄
- [ ] プレースホルダーが表示される
- [ ] 入力開始と同時にプレースホルダーが消える
- [ ] フォーカスが外れた時、空の場合はプレースホルダーが表示される

### モード状態表示
- [ ] 通常モード時、「通常モード」（グレー）が表示される
- [ ] 分析モード時、「分析モード（DB限定）」（ブルー）が表示される
- [ ] 定型レポート実行時、「定型レポート実行中」（グリーン）が表示される

### 入力補助
- [ ] 「レポート：」で始めると、ドロップダウンが表示される
- [ ] 「報告：」で始めると、ドロップダウンが表示される
- [ ] ドロップダウンから選択できる
- [ ] 通常モード入力（数値など）には影響しない

### 初回ガイド
- [ ] 初回起動時のみ、ガイドが表示される
- [ ] 次回以降は表示されない

## 【最終確認】

### エラー処理
- [ ] すべてのエラーが内部ログに記録される
- [ ] ユーザーには業務向けのメッセージのみ表示される
- [ ] アプリが予期せず終了しない

### パフォーマンス
- [ ] 大量データでも応答が返る（タイムアウトしない）
- [ ] UIがフリーズしない

### 互換性
- [ ] Windows 10/11 で動作する
- [ ] EXE化しても動作する
- [ ] 異なる環境でも動作する
