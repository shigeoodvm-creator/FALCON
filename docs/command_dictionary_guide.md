# コマンド辞書の使い方

コマンド辞書（`command_dictionary.json`）では、**登録名**を入力すると**展開コマンド**に置き換えて実行できます。よく使うコマンドを短い名前で登録して、自作の一覧や集計を呼び出せます。

---

## リストコマンドを自作するときの書式

```
リスト：項目1 項目2 項目3 … ：条件 ：並び順
```

- **項目** … 表示したい列をスペース区切りで指定（項目辞書の表示名を使用）
- **条件** … 省略可。フィルタ条件を指定
- **並び順** … 省略可。「昇順 項目名」または「降順 項目名」

コロン（`：` または `:`）で区切った**最大3部分**で、順に「項目」「条件」「並び順」になります。

---

## 並び順の指定（分娩予定日で降順にする例）

リストを**分娩予定日で新しい順（降順）**にしたい場合は、3番目の部分に **「降順 分娩予定日」** を付けます。

- 並び順なし（デフォルトはID昇順）  
  `リスト：ID 受胎日 分娩予定日 RC：RC=5,6`

- **分娩予定日で降順**  
  `リスト：ID 受胎日 分娩予定日 RC：RC=5,6：降順 分娩予定日`

条件がない場合は、並び順だけを2番目に書けます。

- `リスト：ID 分娩予定日：降順 分娩予定日`

---

## 条件の例

| 条件の書き方 | 意味 |
|-------------|------|
| `RC=5,6` | 受胎（繁殖コード 5, 6 のみ） |
| `RC=1,2,3,4` | 空胎 |
| `受胎` | RC=5,6 と同じ |
| `空胎` | RC=1,2,3,4 と同じ |
| `DIM>150` | 分娩後日数が150より大きい |
| `AGE>30` | 日齢が30より大きい |

---

## 分娩予定表の登録例（コマンド辞書）

登録名: **分娩予定表**  
展開コマンド:

```
リスト：ID 受胎日 受胎SIRE 受胎授精種類 予定分娩間隔 分娩予定60日前 分娩予定40日前 分娩予定21日前 分娩予定日 RC：RC=5,6：降順 分娩予定日
```

- 表示列: ID, 受胎日, 受胎SIRE, 受胎授精種類, 予定分娩間隔, 分娩予定60日前, 分娩予定40日前, 分娩予定21日前, 分娩予定日, RC  
- 条件: 受胎のみ（RC=5,6）  
- 並び順: **分娩予定日で降順**（予定日が新しい順）

---

## その他

- 項目名は**項目辞書（item_dictionary.json）の表示名**で指定します。
- リスト実行時、画面上の「並び順」欄で列と昇順/降順を指定しても、コマンド末尾に同じように「：昇順/降順 項目名」が付与されます。
- コマンド辞書は **設定 → コマンド辞書** から編集できます。使い方ボタンでこの説明を参照できます。
